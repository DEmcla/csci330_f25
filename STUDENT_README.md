# CSCI 330: Data Structures in C++ - Student Repository

Welcome to the CSCI 330 course repository! This repository contains all the learning materials you need for the course.

## 📚 What You'll Find Here

### Module Materials (`/modules/`)
Each module directory contains:
- **lecture-notes.md** - Complete lecture content and examples
- **assignments.md** - Programming assignments and exercises
- **quick-help.md** - Quick reference and common solutions
- **tutorial-*.md** - Hands-on tutorials (select modules)
- **debugging-guide.md** - Debugging tips (where applicable)
- **README.md** - Chapter overview and learning objectives

### Assignment Templates (`/assignment-templates/`)
Starter code and test files for homework assignments:
- Each assignment has its own directory (HW_01, HW_02, etc.)
- Includes starter code, test files, and submission guidelines
- Makefile provided for building and testing

### Projects (`/projects/`)
Semester-long project materials:
- Project specifications
- Milestone requirements
- Example implementations (where appropriate)

### Supplemental Materials (`/supplemental/`)
Additional learning resources and advanced topics

## 🚀 Getting Started

1. **Clone the Repository**
   ```bash

   cd csci330_f25
   ```

   ```bash

   ```

3. **Read the Learning Materials**
   - Start with README.md for overview

   - Complete assignments.md
   - Use quick-help.md for reference

## 📖 Course Structure

The course follows a progressive structure:
1. **Foundations** (Ch 1-3): Types, Memory, References
2. **Objects** (Ch 4-6): Lifecycle, Polymorphism, Templates
3. **Language** (Ch 7-9): Expressions, Statements, Functions
4. **Advanced** (Ch 10-12): Smart Pointers, Utilities
5. **STL** (Ch 13-15): Containers, Iterators, Strings
