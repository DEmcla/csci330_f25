# HW_07: Algorithm Analysis - Sorting Algorithms

## Overview
This assignment focuses on implementing and analyzing different sorting algorithms.

## Related Module
📚 **[Module 06 - Chapter 6: Algorithm Analysis and Sorting](../../modules/module-06_ch6/)**

## Learning Objectives
- Implement classic sorting algorithms
- Analyze algorithm performance
- Compare time complexities
- Understand algorithmic efficiency

## Assignment Tasks
Complete the TODOs in `src/main.cpp`:

1. **Algorithm Implementation**: Code bubble, selection, and insertion sort
2. **Performance Testing**: Measure execution times
3. **Data Generation**: Create test datasets of various sizes
4. **Analysis**: Compare algorithms across different scenarios
5. **Bonus**: Add quicksort to the comparison

## Building and Testing
```bash
mkdir build && cd build
cmake ..
make
./sorting_analysis
make test
```
