# HW_03: Vectors and Control Structures

## Overview
This assignment focuses on using STL vectors and implementing various control structures for data manipulation.

## Related Module
📚 **[Module 02 - Chapter 2: Control Structures and Functions](../../modules/module-02_ch2/)**

## Learning Objectives
- Master STL vector operations
- Implement range-based for loops
- Practice search algorithms
- Understand container manipulation

## Assignment Tasks
Complete the TODOs in `src/main.cpp`:

1. **Vector Creation**: Create and populate a vector with student names
2. **Data Display**: Use range-based loops to display all elements
3. **Search Implementation**: Allow user to search for specific names
4. **Index Display**: Show the position of found elements
5. **Bonus**: Implement alphabetical sorting
