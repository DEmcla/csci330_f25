# HW_06: Object-Oriented Programming - Classes

## Overview
This assignment introduces class design, encapsulation, and object-oriented programming principles.

## Related Module
📚 **[Module 05 - Chapter 5: Classes and Objects](../../modules/module-05_ch5/)**

## Learning Objectives
- Design and implement custom classes
- Practice encapsulation and data hiding
- Implement constructors and methods
- Work with collections of objects

## Assignment Tasks
Complete the TODOs in `src/main.cpp`:

1. **Class Design**: Create Student class with private members
2. **Constructor Implementation**: Initialize objects properly
3. **Method Development**: Implement getters, setters, and utility methods
4. **Object Management**: Work with collections of Student objects
5. **Bonus**: Implement sorting by GPA

## Building and Testing
```bash
mkdir build && cd build
cmake ..
make
./student_manager
make test
```
